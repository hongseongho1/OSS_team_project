# KillTime Recommender

한국외국어대학교 Open Source Software 과목 팀프로젝트 과제 입니다.
대학생들을 위한 킬링타임 추천 웹사이트 프로젝트입니다.

## 🧩 구성
- Streamlit 기반 웹페이지
- 영화 추천 시스템
- GitHub 협업 기반 개발

## 📁 디렉토리 구조
- `app/`: 웹 프론트
- `utils/`: 추천 알고리즘 모듈
- `data/`: 영화 데이터셋
- `notebooks/`: 실험용 코드

## ✅ 팀 역할
- 프론트엔드: 장예령
- 추천 시스템: 이동제, 홍성호

## ⛳ 목표 기능
- 장르 선택 기반 추천
- 기본 필터링 알고리즘
- 반응형 UI
